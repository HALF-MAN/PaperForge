# Mission Memory

Use this skill after a meaningful workflow run.

## Instructions

- Extract reusable lessons from run trace and artifacts.
- Link memory to the source mission.
- Avoid promoting sensitive or private execution details by default.

## Deliverable

The deliverable is a `MemoryNote` artifact.
