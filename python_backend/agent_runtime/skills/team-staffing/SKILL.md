# Team Staffing

Use this skill after mission intake.

## Instructions

- Resolve required skills from the mission objective.
- Select agents whose `skillIds` cover the required capabilities.
- Compile a workflow plan with typed inputs, outputs, dependencies, and acceptance criteria.
- Insert policy and human approval gates when the mission touches execution or deployment.

## Deliverable

The deliverables are a `TeamPlan` artifact and a `WorkflowPlan` embedded in the mission.
